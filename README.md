# Blink

Blink *- это простая и настраиваемая начальная страница для вашего браузера с RSS-ридером, построенным на PHP и Bootstrap 3.*

## Возможности

* **Поисковик** с поддержкой Google Programmable Search Engine (CSE)
* **RSS-ридер** для чтения новостных лент
* **Настраиваемые цветовые схемы**: синий, красный, жёлтый, чёрный
* **Управление ссылками** в навбаре (добавление/удаление дополнительных ссылок)
* **Управление RSS-фидами** (добавление множества фидов для быстрого доступа)
* **Сохранение настроек** в cookies (30 дней)

## Установка

1. Скопируйте файлы на ваш веб-сервер с поддержкой PHP
2. Убедитесь, что PHP имеет доступ к `simplexml` для парсинга RSS
3. Откройте `index.php` в браузере

## Настройка

### Цветовая схема

Откройте настройки (иконка шестерёнки справа в навбаре) и выберите цветовую схему:

* Синий (по умолчанию)
* Красный
* Жёлтый
* Чёрный

### Google CSE ID

В настройках можно указать свой Google Custom Search Engine ID (cx).

### Дополнительные ссылки

В настройках можно добавлять дополнительные ссылки в навбар. Кнопки "поисковик" и "feed" нельзя изменить.

### RSS-фиды

На странице `feed.php` можно добавлять множественные RSS-фиды через настройки. Добавленные фиды отображаются как кнопки для быстрого переключения.
